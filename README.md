# Data Science Coursework — Jupyter Notebooks

Python / data science assignments from a university course.
Topics covered: descriptive statistics, inferential statistics, Pandas, and linear regression.

## Notebooks

| Notebook | Topic |
|---|---|
| `Practica_Estadsitica_Descriptiva.ipynb` | Descriptive statistics — measures of central tendency, spread, and distribution |
| `Practica 3 Limpia.ipynb` | Pandas data wrangling — merge, EDA, correlation heatmap, grouped bar charts |
| `Cayol04EstadisticaPandas.ipynb` | Combined stats + Pandas practice — kiosk customer analysis with real-world data |
| `Practica05_Parte_1_Estadistica_Inferencial.ipynb` | Inferential statistics — Central Limit Theorem, confidence intervals, hypothesis testing |
| `Practica05_Parte2_Estadistica_Inferencial.ipynb` | Inferential statistics (continued) |
| `Cayol_REgresion_Lineal.ipynb` | Linear regression on California Housing dataset — EDA, sklearn, PCA visualization |

## Stack

- Python 3.x (Miniconda)
- numpy, pandas, matplotlib, seaborn, scipy, scikit-learn

## Setup

```bash
pip install -r requirements.txt
jupyter notebook
```

## Data

- `GlobalFindexDatabase2025.csv` — World Bank Global Findex dataset (financial inclusion indicators)
- `Glossary_Findex.xlsx` — Variable definitions for the Findex dataset
- `my_data.csv` — Custom dataset used in descriptive statistics exercises
